# 12.7. Домашнее задание

В итоговом домашнем задании вашей задачей будет написать функцию `IsKPeriodic`, которая проверяет, является ли строка кратной числу `K`.

Кратность — повторяемость строки каждые `K` символов (`K` должно быть больше `0`). Например, `abcabcabcabc` имеет кратность `3`,  так как она состоит из подстрок `abc`, имеющих длину `3`.

*Задание загружено в репозиторий на GitHub.*

### Алгоритм
Сначала осуществляется подсчёт числа повторений символов из входной строки `char text[]`, 
учёт происходит в массиве `int textCount[256]`.
Далее анализируем массив `textCount[]` по количеству символов и пытаемся понять к какой группе относится тот или иной символ:
```
int minFind = 0, // Минимальное количество повторений символов (очистка от шума, одиночных посторонних символов)
    avgFind = 0, // Целевое количество повторений символов
    maxFind = 0; // Максимальное количество повторений символов (повторяющихся в паттерне символов)
```
и сравниваем полученный `K = (strlen(text) / avgFind)` с ожидаемым K на входе функции `bool IsKPeriodic(char text[], int K)`
