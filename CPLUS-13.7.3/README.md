# Задание 13.7.3
## Удаление узлов
Метод удаления узлов из дерева мы предлагаем вам реализовать самостоятельно.

Обозначим, что, если у узла есть дочерние вершины, то все эти вершины становятся детьми родителя удаляемого узла. Для упрощения не рассматривайте случай, когда удаляется корневой узел. То есть дерево должно оставаться связным, чтобы между любыми вершинами дерева существовал путь. Пример удаления вершины изображен на рисунке 13.7.7 и 13.7.8. Как меняется дерево после удаления вершины ‘4’ проиллюстрировано ниже. Узлы ‘5’ и ‘6’ становятся дочерними к узлу ‘1’.
